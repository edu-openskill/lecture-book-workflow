# 강의 해부 — 추출 규칙 + 산출물 템플릿

## 판서 대본에서 무엇을 뽑나

판서 대본은 슬라이드별로 "말할 내용 + 그 밑 개념 정의" 구조다. 여기서 4가지를 뽑는다.

1. **개념** — 슬라이드 제목과 "개념 정의" 블록. 강이 가르치려는 핵심 명제.
2. **비유** — 강사가 개념을 풀어낸 일상 비유. 원문 그대로 수집(가공 금지).
3. **오개념** — 강이 "부수려는" 흔한 착각. 보통 "흔히 ~라고 생각하지만 사실은…" 패턴.
4. **선수 개념** — 이 강을 이해하기 위해 먼저 알아야 하는 앞 강의 개념.

## 자막(caption-clean.md)에서 보강할 것

- 대본에 없던 **즉흥 예시·비유의 부연**(강사가 라이브로 더 설명한 부분).
- **표현의 결**(실제 말투) — 책의 대화체 톤을 잡는 참고.
- 필러어("어…","그…","자 이제")는 버린다. 사실/예시만 취한다.

## 의도 필터 (seed.md)

- seed.md의 "이 책이 다루는 범위" 안에 드는 개념만 책에 담는다.
- 강의엔 있어도 책 범위 밖이면 `제외` 표에 사유와 함께 남긴다(나중에 되살릴 수 있게).

## 산출물 템플릿: planning/lecture-analysis.md

```markdown
# 강의 해부

## 소스 목록
| 강 | 판서대본 | 자막 | 비고 |
|----|---------|------|------|
| 1강 | O | O | |
| … | | | |

## 강별 개념 맵 (의도 안)
### N강 — [제목]
- 핵심 메시지(한 줄):
- 개념:
  - [개념명] — [한 줄 정의] (등장: 슬라이드 X)
- 비유(강사 원문):
  - [비유] = [개념]
- 오개념(부술 착각):
  - 착각: … → 교정: …
- 선수 개념: (N강은 M강의 □□ 개념을 전제)

## 의도 밖 (제외)
| 내용 | 출처 강 | 제외 이유 |
|------|--------|----------|
| … | … | seed.md 범위 밖 |

## 강 간 의존성 그래프
(예: 5강 화살표 → 6강 벡터 → 9강 컨텍스트크기 …)

## 책 구성에 줄 함의 (STEP 3로 넘길 메모)
- 묶을 후보: (예: 9~11강은 "컨텍스트 한계" 한 부로)
- 순서 주의: (예: 어텐션(6강)은 17강 에이전트보다 반드시 앞)
```

## 검토 포인트 (인사이트 질문 예시)

- "5강 화살표 비유가 6강 벡터에서 다시 쓰이는데, 책에서도 같은 비유로 이어갈까?"
- "이 강은 오개념을 안 짚고 넘어가는데, 책 도입을 어떤 문제로 열까?"
- "이 개념은 강의 본편엔 없고 자막 즉흥 설명에만 있어 — 책에 넣을까?"
