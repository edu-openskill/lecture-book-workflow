# 자막 정제 규칙 + 용어 교정 사전

## Cap2. VTT 정제 (caption-raw.vtt -> 평문)

제거할 것:
- `WEBVTT` 헤더, `Kind:`/`Language:` 메타행
- 타임코드 줄 (`00:00:03.000 --> 00:00:05.000`)
- 인라인 타이밍 태그 `<00:00:03.000>`, `<c>...</c>`
- 유튜브 자동자막 특유의 **중복행**(다음 줄과 겹쳐 한 줄씩 밀려 반복되는 패턴) — 연속 중복 제거
- 빈 줄 다중 -> 한 줄

복원할 것:
- 끊긴 조각을 문장 단위로 이어 붙임(마침표·물음표·의문 어미 기준)
- 화면 줄바꿈으로 쪼개진 한 문장을 한 줄로

버릴 것(필러어):
- "어", "음", "그", "자", "이제", "뭐", "인제", "약간" 등 의미 없는 추임새
- 같은 말 반복("그러니까 그러니까")

## Cap3. 용어 교정 사전

유튜브 한국어 자동자막이 자주 틀리는 도메인 용어. 좌측(오인식) -> 우측(정정):

| 오인식 예 | 정정 |
|----------|------|
| 컨텍스, 콘텍스트, 컨텍슽 | 컨텍스트 |
| 어텐션을, 애텐션, 어텐숀 | 어텐션 |
| 라그, 래그, 알에이지 | RAG |
| 엠씨피, 엠시피 | MCP |
| 엘엘엠, 엘렐엠 | LLM |
| 임베딩, 임배딩 | 임베딩 |
| 백터, 벡타 | 벡터 |
| 할루시네이션, 헐루시네이션, 환각 | 할루시네이션 |
| 토큰화, 토크나이즈 | 토큰 / 토크나이즈 |
| 에이전트, 에이젼트 | 에이전트 |
| 제이아이티, 지트 | JIT |
| 파인튜닝, 파인튜인 | 파인튜닝 |

> 사전은 프로젝트마다 추가 가능. 새 강의에서 반복되는 오인식이 보이면 여기에 누적한다.

## Cap4. 대본 정렬 (정렬표.md)

판서대본의 슬라이드 제목(`##`, `###` 헤딩)을 앵커로 삼아, 자막 구간을 슬라이드에 매핑한다.

산출물 템플릿:

```markdown
# N강 자막↔대본 정렬표

| 슬라이드(대본 헤딩) | 자막 구간 요지 | 대본에 없던 보강 포인트 |
|--------------------|---------------|------------------------|
| §1 오프닝 | … | (즉흥으로 든 예시 메모) |
| §2 개념 도입 | … | |
```

정렬이 애매하면 시간순으로 두고 "보강 포인트"만 표시한다. 완벽한 매핑보다 **놓친 디테일 회수**가 목적.
