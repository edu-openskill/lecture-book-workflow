# 책별 핵심 비유 화이트리스트 — 템플릿

이 파일을 `projects/<책>/planning/prose-polish-allowlist.md` 로 복사해 책마다 채웁니다.

prose-polish 스킬은 이 파일을 로드하여 *처음 도입 시점에서 살릴 비유*를 식별합니다.

## 작성 방법

각 항목에 다음을 기록합니다.

- **비유어**: 본문에 등장하는 비유 단어
- **가리키는 개념**: 어떤 정식 용어를 대신하는지
- **도입 위치**: 책에서 *처음 도입*된 챕터·절 (이 위치에서는 무조건 유지)
- **회수 허용**: 이후 챕터에서 *반복 회수* 시 어디까지 허용하는지

## 항목 형식

```
### 비유어
- **가리키는 개념**: 
- **도입 위치**: 
- **회수 허용**: 
- **금지 변형**: (있으면)
```

## 예시 (이 책 — 특이점이-온-개발자-도커-쿠버네티스)

### 컨테이너 = 상자
- **가리키는 개념**: 컨테이너(Container)
- **도입 위치**: CH01 1.2 (말콤 맥린·항구 이야기)
- **회수 허용**: 전 챕터. 단 *수사적 변형*("그릇·통·박스·캐비넷")은 금지
- **금지 변형**: 그릇, 통, 박스, 캐비넷

### Docker = 공장 / 이미지 = 설계도
- **가리키는 개념**: Docker, 이미지(Image)
- **도입 위치**: CH01 1.3
- **회수 허용**: 전 챕터

### Kubernetes = 항만 관제 센터
- **가리키는 개념**: Kubernetes
- **도입 위치**: CH01 1.3
- **회수 허용**: 전 챕터

### 가맹점·본사·매장
- **가리키는 개념**: 클러스터·컨트롤플레인·노드/Pod
- **도입 위치**: CH04 4.1.2 (선배 대사로 도입)
- **회수 허용**: CH04~CH06

### 공용 메뉴판 / 비밀 레시피
- **가리키는 개념**: ConfigMap / Secret
- **도입 위치**: CH06 6.1.1
- **회수 허용**: CH06 한정

### 주방 / 요리사
- **가리키는 개념**: 호스트·가상화·컨테이너
- **도입 위치**: CH02 2.1
- **회수 허용**: CH02 한정 (이후 챕터에서 호출 시 *처음 도입* 위치를 참조하는 식)
