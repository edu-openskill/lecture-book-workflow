# CLAUDE.md 거버넌스

CLAUDE.md를 수정할 때 적용되는 규칙.

## 200줄 제한

- CLAUDE.md는 **200줄 이내**로 유지한다
- 수정 완료 후 `wc -l CLAUDE.md`로 줄 수를 확인한다
- 200줄 초과 시 상세 내용을 분리하고 참조만 남긴다

## 분리 전략

상세 내용을 넣을 곳이 이미 있으면 그곳으로 보낸다.

| 내용 유형 | 보낼 곳 | CLAUDE.md에 남길 것 |
|----------|---------|-------------------|
| 에이전트 실행 상세 | `.claude/agents/*/AGENT.md` | 1줄 요약 + `상세: AGENT.md` |
| 스킬 사용법 상세 | `.claude/skills/*/SKILL.md` | 실행 명령 1줄 + `상세: SKILL.md` |
| 글쓰기 규칙 | `.claude/rules/*.md` | 규칙 테이블 한 줄 |
| 워크플로우 상세 | `.claude/workflow/*.md` | 명령어 테이블 한 줄 |
| 기타 참조 문서 | `docs/` | `상세: docs/파일.md` |

## 임포트 패턴

CLAUDE.md에서 다른 파일을 참조할 때:
- 규칙 파일: `.claude/rules/`에 넣으면 자동 로드 (임포트 불필요)
- 참조 문서: `@docs/파일.md`로 임포트 가능 (세션 시작 시 로드)
- 에이전트/스킬: 해당 에이전트 호출 시 자동 로드 (CLAUDE.md에서 임포트 불필요)
