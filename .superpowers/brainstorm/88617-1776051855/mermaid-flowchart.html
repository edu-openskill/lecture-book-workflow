사용 가이드
  • Notion: /mermaid 블록에 코드 붙여넣기
  • GitHub: 이슈·PR·README에서 ```mermaid ... ``` 코드 블록 사용
  • Obsidian: 코드 블록 언어를 mermaid로 지정
  • mermaid.live: mermaid.live에 코드 붙여넣고 PNG/SVG 내보내기
  • 기술서적: Markdown에서 mermaid 코드블록 → pub-build 파이프라인이 PNG 렌더링
