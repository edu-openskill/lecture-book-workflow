# 5.7 Postman 실습 2

업로드 후 3~5초 대기한 뒤 완료 저장 API를 호출해 DB 저장을 검증합니다. 서버를 다시 실행하여 /presigned요청하여 presignUrl을 발급 받은 후 업로드를 완료한 후 진행합니다.

---

## 1) resized 폴더 확인

![image.png](5%207%20Postman%20%EC%8B%A4%EC%8A%B5%202/image.png)

발급 받은 presignUrl로 PUT요청을 성공하였다면 original폴더로 업로드 된 후 Lambda에서 트리거 되어 설정해 놓은 리사이즈 함수가 적용되어 resized 폴더에 저장 되었다는 걸 확인할 수 있습니다.

리사이즈가 완료되었으며 해당 경로를 서버에 저장하는 요청을 보냅니다.

---

## **2) 완료 저장 요청**

![image.png](5%207%20Postman%20%EC%8B%A4%EC%8A%B5%202/image%201.png)

`POST http://localhost:8080/complete`

key 값은 /presigned 요청해서 응답 받았던 key값을 복사하여 붙여넣습니다.

## **3) 응답 확인**

![image.png](5%207%20Postman%20%EC%8B%A4%EC%8A%B5%202/image%202.png)
