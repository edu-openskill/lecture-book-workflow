## **1) 서비스에 조회 메서드 추가**

**(입력) 경로: src/main/java/com/metacoding/spring_presign_url/image/ImageService.java**

```java
public ImageResponse.Items listAll() {
    List<ImageResponse.Item> items = imageRepository.findAll()
            .stream()
            .map(ImageResponse.Item::fromEntity)
            .collect(Collectors.toList());
    return new ImageResponse.Items(items);
}

public ImageResponse.Detail findById(Long id) {
    ImageEntity entity = imageRepository.findById(id)
            .orElseThrow(() -> new RuntimeException("이미지를 찾을 수 없습니다."));
    return ImageResponse.Detail.fromEntity(entity);
}
```

전체 조회와 단건 조회 서비스를 제공합니다.

## **3) 응답 DTO (Items/Detail)**

목록과 상세보기 응답 DTO를 정의합니다.

**(확인) 경로: src/main/java/com/metacoding/spring_presign_url/image/ImageResponse.java**

```java
public class ImageResponse {
    public record Item(
            Long id,
            String originalUrl,
            String resizedUrl) {
        public static Item fromEntity(ImageEntity imageEntity) {
            return new Item(
                    imageEntity.getId(),
                    imageEntity.getOriginalUrl(),
                    imageEntity.getResizedUrl());
        }
    }

    public record Items(List<Item> items) {
    }

    public record Detail(
            Long id,
            String uuid,
            String originalUrl,
            String resizedUrl,
            String fileName,
            @JsonFormat(pattern = "yyyy-MM-dd HH:mm") LocalDateTime createdAt) {
        public static Detail fromEntity(ImageEntity imageEntity) {
            return new Detail(
                    imageEntity.getId(),
                    imageEntity.getUuid(),
                    imageEntity.getOriginalUrl(),
                    imageEntity.getResizedUrl(),
                    imageEntity.getFileName(),
                    imageEntity.getCreatedAt());
        }
    }
}
```

목록은 `items`, 상세는 `detail`로 고정합니다. 목록은 가벼운 필드만 반환합니다.
