### 2.2 S3 권한

![image.png](5%201%20%EC%A4%80%EB%B9%84/image%205.png)

생성 버킷 리스트를 확인하고 객체생성 및 권한설정을 위하여 생성한 버킷을 클릭합니다.

---

![image.png](5%201%20%EC%A4%80%EB%B9%84/image%206.png)

폴더 만들기를 클릭하여 원본 이미지를 저장하는 orignal 폴더, 리사이즈 이미지를 저장하는 resized 두가지 폴더를 생성합니다.

---

![image.png](5%201%20%EC%A4%80%EB%B9%84/image%207.png)

```java
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicAllAccess",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:*",
      "Resource": [
        "arn:aws:s3:::metacoding-base64",
        "arn:aws:s3:::metacoding-base64/*"
      ]
    }
  ]
}
```

권한 탭으로 이동하여 버킷 정책에서 편집을 클릭하여 json을 입력하고 변경사항을 저장합니다.

## **2) 환경변수 작성**

IAM에서 다운로드한 ACCESS_KEY와 SECRET_KEY를 아래 포멧에 맞춰 .env 파일을 생성합니다.

**(입력) 경로: .env**

```java
# ---------------------------
# AWS 환경변수
# ---------------------------
CLOUD_AWS_CREDENTIALS_ACCESS_KEY="ACCESS_KEY"
CLOUD_AWS_CREDENTIALS_SECRET_KEY="SECRET_KEY"
CLOUD_AWS_REGION=ap-northeast-2
CLOUD_AWS_S3_BUCKET="S3_BUCKET_NAME"
```

이 파일에 실습에 필요한 AWS/Spring 환경변수를 정의합니다.
