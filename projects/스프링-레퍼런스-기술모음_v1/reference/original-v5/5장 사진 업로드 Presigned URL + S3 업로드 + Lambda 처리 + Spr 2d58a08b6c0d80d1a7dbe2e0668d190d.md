# 5장 사진 업로드 Presigned URL + S3 업로드 + Lambda 처리 + Spring 저장 (Postman 테스트)

이번 챕터에서는 Spring 서버를 실행한 뒤 Presigned URL로 AWS S3에 업로드하고, Lambda로 리사이즈한 결과를 저장한 다음, Spring에서 resized URL을 저장·조회하는 흐름을 실습합니다.

이 실습은 **start 코드(기본 환경**)에서 시작해 **end 코드(단계별 커밋 포함)**를 따라가며 완성하는 방식으로 진행합니다. 각 단계에서는 안내된 커밋을 기준으로 변경된 파일(diff)을 확인하면서 그대로 적용하면 됩니다.
