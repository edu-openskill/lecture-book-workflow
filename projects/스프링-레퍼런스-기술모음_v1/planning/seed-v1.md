# 씨앗 v1 — 스프링과 함께 하는 레퍼런스 기술 모음

## 이 책은 뭐다

스프링 생태계의 **9개 실전 레퍼런스 기술**을 한 서비스에 얹어 가는 책. 단일 Spring Boot 애플리케이션이 실제 기업 환경에서 마주치는 인증·캐시·파일·이벤트·검색·스트리밍을 거쳐 **성장 여정**을 따라 움직인다. 조각난 예제가 아니라, 하나의 서비스를 9단계로 키우는 이야기.

## 독자

Spring Boot 기본 문법과 REST API는 이해했지만, Redis·S3·Elasticsearch·RabbitMQ 같은 **주변 인프라를 처음 붙여 보는 주니어/중급 백엔드 개발자**. 강의에서 이름은 들어봤는데, 실제로 왜 이걸 끼우는지 감이 오지 않는 단계.

## 톤·구조 상수

- 이야기 기반 · 통합 흐름 · 캐릭터 삼각 구도
- 주인공 **오픈이** · **팀장**(힌트 제공자) · **동료**(문제 제기자)
- Try/Fail 루프 최소 1회 · 실패부터 시작
- 비유 → 정식 정의 2단계
- 실습 챕터 12단계 포맷 (`.claude/rules/chapter-format.md` 참조)
- 문체: `~합니다` / `~입니다` (`style.md`)
- 이모지 · 수평선 · 라벨형 H2 금지

## 기술 원칙

- 원문 v5 9장의 **실전 예제 코드는 그대로 유지**. 문체와 서사만 각색
- 각 챕터는 **직전 챕터 위에 누적**되어 서비스가 단계적으로 성장
- 이야기 파트에는 코드 없음. 기술 파트에서만 코드 등장

## 9장 매핑 (원문 v5 → 이 책)

1. Docker로 시작하는 개발 환경
2. OAuth2.0 & OIDC — 카카오 로그인
3. Redis 세션 공유
4. Spring 이미지 업로드 — Base64
5. 대용량 업로드 — Presigned URL + S3 + Lambda
6. Polling · SSE · WebSocket 실전 비교
7. 대규모 스트리밍 실전과 기술 선택
8. Elasticsearch 키워드 검색
9. RabbitMQ 파일 변경 감지 → 발행·구독 자동 동기화
