## 2. 추가정보

### 책소개

하나의 쇼핑몰 프로젝트를 다섯 단계에 걸쳐 모놀리식에서 마이크로서비스로 전환하며, MSA의 핵심 구조를 직접 만들어 보는 실습서다.

챕터 1에서 모놀리식의 한계와 MSA의 필요성을 짚은 뒤, 챕터 2에서 Spring Boot 기반 4개 서비스(회원·상품·주문·배달)를 REST로 연결하고 보상 트랜잭션을 직접 구현한다. 챕터 3에서는 UseCase 인터페이스를 도입해 클린 아키텍처로 리팩토링하고, Docker 이미지를 빌드하여 Kubernetes(Minikube)에 배포한다. 챕터 4에서 Kafka를 도입해 동기 호출을 비동기 메시지로 전환하고, Orchestration Saga 패턴으로 분산 트랜잭션을 조율한다. 챕터 5에서는 WebSocket(STOMP)을 붙여 주문·배달 완료를 실시간으로 사용자에게 Push한다.

이 책을 통해 배우는 내용은 다음과 같다.

- Spring Boot로 독립된 마이크로서비스 4개를 설계하고 연결하는 방법
- REST 동기 통신에서 보상 트랜잭션을 직접 구현하는 방법
- UseCase 인터페이스를 활용한 클린 아키텍처 리팩토링
- Docker 이미지 빌드와 Kubernetes 매니페스트 작성 및 배포
- Kafka를 활용한 비동기 메시지 통신 구조
- Orchestration Saga 패턴으로 분산 트랜잭션을 조율하는 방법
- WebSocket(STOMP)으로 실시간 알림을 구현하는 방법

이런 분께 추천한다.

- Spring Boot 경험은 있지만 MSA를 처음 접하는 백엔드 개발자
- MSA 개념은 알지만 코드로 구현해 본 적 없는 개발자
- Kafka, Kubernetes를 실제 프로젝트에 적용해 보고 싶은 개발자
- 분산 트랜잭션과 Saga 패턴을 직접 만들어 보고 싶은 개발자
