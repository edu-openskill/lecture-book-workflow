# 가독성 분석 리포트

5개 챕터(총 4,099라인)를 한 단락씩 정밀 Read 하여 가독성이 떨어지는 문장·단락을 식별. 비유나 캐릭터 대사가 의도적으로 배치된 이야기 파트는 제외. `~요` 종결은 별도 처리 대상이므로 본 분석에서 제외.

분류 기준:
- A. 흐름 끊김 (비유가 사실-이유 사이에 끼어듦)
- B. 문장 길이·압축 (100자+, 정보 3개+, 쉼표 5개+)
- C. 연결성 부족 (접속어 부재로 점프)
- D. 어색한 종결·문체 (명사형 종결, 한 문장 능동/수동 혼재)
- E. 정보 순서 어색 (결론·부연 순서, 비유-정의 어긋남)
- F. 명사 나열·번역투 ("X의 Y의 Z" 3단 소유격, 수동 번역투)
