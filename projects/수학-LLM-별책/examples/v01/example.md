# 1강 계산 예제 — 2차방정식과 최솟값

## 예제 1: 포물선의 가장 낮은 점
**문제**: $f(x) = x^2 - 4x + 7$ 의 최솟값과 그때의 $x$를 구하라.

**풀이 (완전제곱식)**:
1. 일차항 계수 $-4$의 절반 $-2$로 묶는다 — $f(x) = (x^2 - 4x + 4) + 3$
2. 완전제곱 — $f(x) = (x-2)^2 + 3$
3. $(x-2)^2 \ge 0$ 이므로 가장 작아질 때는 $(x-2)^2 = 0$, 즉 $x = 2$.

**검산 (꼭짓점 공식)**: $x = -\dfrac{b}{2a} = -\dfrac{-4}{2\cdot 1} = 2$, $f(2) = 4 - 8 + 7 = 3$.

**답**: 최솟값 $3$ (이때 $x = 2$).

**왜 이게 AI에 쓰이나**: "학습 = 손실(오차)을 가장 작게"의 가장 단순한 원형. 최솟값을 *찾는다*는 감각을 여기서 처음 심는다. (→ 9강 경사하강법, L4 MSE)
