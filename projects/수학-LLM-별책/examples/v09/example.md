# 9강 계산 예제 — 경사하강법

## 예제 1: 기울기를 따라 골짜기로 내려가기
**문제**: $f(x) = x^2 - 4x + 7$ 을 경사하강법으로 최소화한다. 시작점 $x_0 = 0$, 학습률 $\eta = 0.1$ 로 두 걸음 내려가라. (이 함수의 진짜 최솟값은 1강에서 구한 $x=2$.)

**풀이**:
1. 기울기 함수 — $f'(x) = 2x - 4$  *(6강 미분)*
2. 갱신식 — $x_{n+1} = x_n - \eta\, f'(x_n)$
3. 1걸음 — $x_1 = 0 - 0.1\,(2\cdot 0 - 4) = 0 - 0.1(-4) = 0.4$
4. 2걸음 — $x_2 = 0.4 - 0.1\,(2\cdot 0.4 - 4) = 0.4 - 0.1(-3.2) = 0.72$

**답**: $x_0=0 \to x_1=0.4 \to x_2=0.72$ — 진짜 최솟값 $x=2$ 를 향해 조금씩 수렴한다.

**왜 이게 AI에 쓰이나**: **AI가 '배우는' 바로 그 알고리즘.** 손실의 기울기 반대 방향으로 가중치를 조금씩 옮긴다 — 1강(최솟값)+6강(미분)의 결합. (→ L8 역전파)
