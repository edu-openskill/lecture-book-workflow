# 개념 해부 — 수학·LLM 별책

> 의도 필터: **"필요한 만큼만, 대신 직접 풀어서 확실히"** (seed-v1.md)
> 칼질 반영: 5강 복소→4강 합본 / 11강 오일러→칼럼 / 13-3·13-4 삭제

## 강별 핵심 개념 — 챕터 Ⅰ (수학, 11강 + 칼럼)

| 강 | 핵심 개념(하나) | 비유 후크 | 정식 정의(요약) |
|----|----------------|----------|----------------|
| 1 · 2차방정식과 최솟값 | 포물선 꼭짓점 = 최솟값 | 공 던지면 그리는 포물선, 가장 낮은 점 하나 | f(x)=ax²+bx+c, 꼭짓점 x=−b/2a |
| 2 · 지수·제곱근 | 거듭제곱/루트/분수지수 | 종이 반복해 접기(폭발적 성장) | aⁿ, a^½=√a, 지수법칙 |
| 3 · 로그 | 지수의 역연산, 곱→합 | 지진·dB·별 밝기(곱을 로그로 잰다) | logₐx, log(xy)=log x+log y |
| 4 · 삼각함수 + 복소평면(합본) | 각을 길이로 / 단위원=회전 | 그림자로 해 높이 재기, 시계바늘 한 바퀴, ×i=90°회전 | sin·cos·tan, (cosθ,sinθ), 복소수 곱=회전 |
| 5 · 함수·극한·기울기 | 한 점에서의 기울기, 극한 직관 | 언덕 경사: 두 점 평균경사 → 붙이면 그 점의 경사 | 평균변화율 → 순간변화율 lim |
| 6 · 미분 | 기울기 함수, f'(x)=0 = 최솟값 후보 | 산에서 발밑 경사 0 = 골짜기 바닥 | 다항식 미분, 극값 판정 |
| 7 · e·ln | eˣ는 미분해도 자기 자신 | 쉴 새 없는 복리의 한계점 e | e≈2.718, ln=logₑ |
| 8 · 체인룰 | 합성함수 미분 = 바깥×안쪽 | 톱니바퀴 연쇄의 비율 곱 | (f∘g)'=f'(g)·g' |
| 9 · 경사하강법 | 기울기 반대로 조금씩, 학습률 | 안개 낀 산, 발밑 경사만 보고 하산 | θ ← θ − η·∇L |
| 10 · 행렬 | 곱셈·내적, 행렬=변환 | 정해진 배합표로 한 번에 섞는 믹서 | 행렬곱, dot product |
| 11 · 벡터·기저·정사영·PCA | 방향·그림자·가장 잘 퍼진 축 | 손전등 그림자=정사영, 가장 길게=제1주성분 | 내적, 정사영, 주성분 |
| 칼럼 · 오일러 항등식 | 조각들이 하나로 만난다(통합) | 수학 다섯 거물의 단체 사진 | e^{iπ}+1=0 (쉬어가는 칼럼) |

## 강별 핵심 개념 — 챕터 Ⅱ (LLM, 11소강)

| 소강 | 핵심 개념 | 선수(쓰는 수학) |
|------|----------|----------------|
| 13-1 확률·정규분포 | 분포, 종 모양의 의미 | 7강 e |
| 13-2 평균·분산·표준편차, n−1 | 흩어짐 재기, 자유도 | 2강 제곱·루트 |
| 13-5 공분산·상관 | 두 변수가 함께 움직이나 | 11강 벡터/PCA |
| 13-6 MSE | 손실을 제곱으로 재기 | 1강 2차식, 9강 경사하강법 |
| 13-7 과적합·과소적합 | 외운 것 vs 이해한 것, 일반화 | — |
| 14-1 선형결합 y=wx+b | 뉴런 하나 = 가중치·편향 | 10강 행렬곱 |
| 14-2 활성화함수 | 직선을 곡선으로(비선형성) | 7강 e, 3강 로그 |
| 14-3 층 구조 | 입력·은닉·출력, (선형결합→활성화) | 14-1 |

| 14-6 어텐션·트랜스포머 | 단어들이 내적으로 서로 보기 | 11강 벡터 내적 + 4강 위치인코딩 |

## 의존성 맵 (수학 → LLM, 목차 흡수)
| 수학 강 | 살아나는 LLM 소강 |
|---|---|
| 1 2차식 최솟값 | 13-6 MSE · 14-5 역전파 |
| 2 제곱·루트 | 13-2 표준편차 · 거리계산 |
| 3 로그 | 13-1 분포 · 14-2 softmax |
| 4 삼각(+복소) | 14-6 위치 인코딩 |
| 7 e·ln | 13-1 정규분포 · 14-2 시그모이드 |
| 8 체인룰 | **14-5 역전파(핵심)** |
| 9 경사하강법 | **14-5 학습 그 자체** |

## 의도 밖 가지치기
| 주제 | 처리 | 이유 |
|------|------|------|
| 11강 오일러 항등식 | 칼럼화 | AI 직접 쓰임 없음(통합의 쾌감용), 잎사귀 노드 |
| 5강 복소평면(독립) | 4강에 합본 | 주 용도가 오일러 무대 깔기 |
| 13-3 모집단 vs 표본 | 삭제 | 통계 일반론, LLM 직결도 낮음 |
| 13-4 신뢰구간·p-value | 삭제 | 연구방법론/평가, 핵심 경로 밖 |
| ε-δ 엄밀 극한 증명 | 제외 | 직관(5강)으로 충분, "직접 풀이" 부담만 키움 |
| 측도론·해석학 증명 디테일 | 제외 | 의도("필요한 만큼만") 밖 |

## 검토 메모
- **인사이트**: 13-7 과적합은 수학 선수가 없어 "이야기 위주" 강이 됨 → 풀이 분량이 적을 수 있으니 13-6 MSE와 묶어 한 강으로 합치는 것도 STEP 3에서 검토.
- **감수(독자 대변인)**: 첫 강(2차식)은 난이도 낮아 OK. 14-5 역전파가 최고 난도 → 8·9·13-6을 반드시 먼저 깔아야(의존성 지켜짐).
