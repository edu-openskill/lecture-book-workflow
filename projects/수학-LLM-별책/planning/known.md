# 독자 지식 원장 (known.md)

> **이 파일의 정체**: "지금 독자가 어디까지 알고 있는가"의 단일 진실 소스(SSOT).
> `progress.json`이 *집필 상태*를 추적하듯, 이 파일은 *독자의 누적 지식*을 추적한다.
> 트랙: concept · 기준 outline: `planning/`의 최신 `outline-v*.md`(가장 높은 번호 = 최신본. 현재 outline-v3)

---

## 사용 규칙 (집필·검증 양쪽이 이 파일을 읽는다)

**① 챕터 쓰기 전 (STEP 5 진입 시)**
- 이 파일의 "현재 최전선"과 "아직 모르는 것"을 먼저 읽는다.
- 새 챕터에서 쓰려는 용어·기호·개념을 점검한다:
  - **known(아래 원장에 있음)** → 그냥 써도 됨. 재설명 불필요.
  - **모름인데 이 챕터의 핵심** → 반드시 *이 챕터 안에서* 비유→정의→유도로 처음부터 설명.
  - **모름인데 곁가지** → 쓰지 말 것(또는 이름만, "OO강에서 자세히").
  - **outline상 뒤 챕터 것** → 절대 당겨 쓰지 말 것. 당겨야만 하면 그건 outline 순서 문제 → 저자에게 보고.

**② 챕터 쓴 뒤 (검증 패스)**
- "독자 검증" 체크리스트(아래)로 위반을 기계적으로 잡는다.
- 통과하면 이 파일 하단 **원장에 그 챕터 줄을 추가**한다(독자가 새로 알게 된 것).

**③ 갱신 주체**: 챕터가 `pass` 되는 순간, 그 챕터가 가르친 개념·기호를 원장에 push 한다. 절대 소급해서 지우지 않는다(append-only).

## 독자 검증 체크리스트 (나) 섹션 = 검토 모드 발동 질문)

각 챕터 완성 후 아래를 묻는다. 하나라도 ✗면 수정.

```
□ [정보충족] 내용에 필요한 정보가 다 있으면서 쉬운가?
□ [순차이해] 한 줄씩 따라가며 하나씩 이해되는가? (점프 없는가)
□ [목적도달] 초보자가 천천히 읽으면 이 장의 목적에 도달하는가?
□ [유도완결] 수식의 유도과정이 있는가? "이게 됐으니 다음은 이렇고…"
            절대 "다 안다고 치고" 중간을 생략하지 않았는가?
□ [생략점검] 짧게 쓰려다 꼭 필요한 내용을 빠뜨리지 않았는가?
□ [사실검증] 오늘 날짜 기준 자료(논문·인터넷)로 내용이 맞는지 검증했는가?
            더 쉽게 설명한 글이 있으면 참고했는가?
□ [계산이미지] 계산 예제에 이미지를 붙이면 더 쉬운데, 할 수 있는 걸 안 한 곳은 없는가?
□ [어려움시각화] 어려운 부분에 이미지를 붙여 시각화했는가?
□ [선수위반] ★ 이 챕터가 known.md에 없는 용어·기호를 정의 없이 썼는가?
            (썼다면 즉시 위반. 정의를 넣거나 빼라)
□ [미래당김] ★ outline상 뒤 챕터의 개념을 미리 꺼내 쓰지 않았는가?
```

> ★ 두 줄이 핵심 — "편미분이 Ch.8에 정의 없이 등장" 같은 사고를 잡는 줄이다.

## 출발선 (Ch.0, 책 시작 전 가정)

독자 상수: "배경지식 있음, 이 책의 주제(AI 수학)만 모름."
- **안다고 가정**: 사칙연산, 변수와 식, 좌표평면(x-y), 부등호, 분수·소수.
- **모른다고 가정**: 이 책이 가르치는 모든 것(아래 원장이 0줄에서 시작).

> **Ch.19 미지로 남긴 것 (의도적)**: ∂·∇·θ갱신·역전파 첫 정식 — **8강에서 비유로 남긴 ∇L·θ식이 여기서 정식 회수 완료**(8강↔19강 "직관→정식" 분리 닫힘). 다층 신경망의 층별 체인(여러 층 ∂ 전파)은 *원리는 1뉴런으로 충분*히 보였고 깊은 망 전체 유도는 범위 밖(직관까지). 활성화 미분·크로스엔트로피 미분도 범위 밖. **어텐션은 Ch.20**. 이제 1~19강 도구 전부 사용 가능.

> **Ch.18 미지로 남긴 것 (의도적)**: 층·순전파 첫 정식(입력→출력 순방향만, ∂·∇ 미사용). **역전파(거꾸로 오차 거슬러 가중치 고치기)·편미분 ∂·그래디언트 ∇·θ 갱신은 전부 Ch.19**(18강은 "역전파는 19강"이라 예고만). 가중치 초기화=정규분포(12강) 이름만.

> **Ch.17 미지로 남긴 것 (의도적)**: 시그모이드·ReLU·softmax 첫 정식(순방향 값 계산만). **층·순전파는 Ch.18**. **활성화함수의 미분(역전파에서 쓰임)·softmax+크로스엔트로피 미분은 Ch.19/범위밖**(17강은 ∂·∇ 미사용). softmax는 12강 확률분포를 만드는 장치로만, 어텐션 가중치화는 Ch.20.

> **Ch.16 미지로 남긴 것 (의도적)**: 뉴런·가중치·편향·선형결합 첫 정식. **활성화함수(곡선·비선형)는 Ch.17**(16강 뉴런은 직선뿐 — "직선만 쌓으면 직선"이 17강 동기). **$w,b$를 학습으로 갱신하는 법(편미분·역전파)은 Ch.19**(16강은 순방향 출력 계산만, ∂·∇ 미사용). **층·순전파는 Ch.18**(16강은 한 층 $W\vec x+\vec b$ 식만 소개). 

> **Ch.15 미지로 남긴 것 (의도적)**: $\hat y$·손실·MSE 첫 정식. **크로스엔트로피는 더알아보기로 이름만**(LLM 실손실이나 MSE로 구조 충분, 풀이는 MSE). **여러 손잡이($w,b$…)의 손실 미분 = 편미분 ∂·그래디언트 ∇·반복 갱신은 전부 Ch.19**(15강은 손잡이 $w$ *하나*짜리 토이 $\hat y=wx$로 포물선만, ∂·∇ 미사용). **뉴런 $y=wx+b$ 정식은 Ch.16**(15강은 $\hat y=wx$ 토이만). 

> **Ch.14 미지로 남긴 것 (의도적)**: 공분산=편차벡터 내적÷n, 상관=코사인유사도(11강 재회)로만 — **공분산행렬·PCA는 본서 범위 밖**(향후 ML/DL 책). **어텐션의 내적 점수는 Ch.20**(여기선 "관계를 잰다=내적"의 통계판까지만). 상관≠인과 명시. ∂·∇ 여전히 미사용.

> **Ch.13 미지로 남긴 것 (의도적)**: 새 기호 $\sum$(합) 도입. **두 변수의 관계(공분산·상관)는 Ch.14** — Ch.13은 변수 *하나*의 중심·흩어짐까지만. **MSE(제곱오차 손실)는 Ch.15**(여기 제곱은 '흩어짐 재기'용이지 '손실'이 아님 — 혼동 주의). 배치정규화·정규화는 이름만 예고. ∂·∇ 여전히 미사용.

> **Ch.12 미지로 남긴 것 (의도적)**: **μ·σ를 데이터에서 계산하는 정식(평균·분산·표준편차 공식, 자유도 $n{-}1$)은 Ch.13** — Ch.12는 μ(중심)·σ(너비)를 *주어진 파라미터*로만 쓰고 이름·기호만 도입(종을 읽고 68-95-99.7로 확률 계산까지). 정규분포 식 $\frac{1}{\sigma\sqrt{2\pi}}e^{-(x-\mu)^2/2\sigma^2}$ 은 **읽을 수만** 있고 정규화상수 유도·적분은 범위 밖(68-95-99.7 비율의 적분 증명도 *결과만* 수용). **공분산·상관 Ch.14, softmax Ch.17, 가중치 초기화 Ch.16+, 크로스엔트로피 Ch.15 더알아보기** — 전부 이름만 예고. ∂·∇ 여전히 미사용. → Ch.13이 μ·σ의 첫 정식 계산.

> **Ch.11 미지로 남긴 것 (의도적)**: **어텐션 정식은 Ch.20** — Ch.11은 내적·코사인 유사도까지만, "내적으로 단어들이 서로 본다(어텐션)"는 *이름만 예고*. **임베딩을 어떻게 만드나·공분산/상관(Ch.14)·PCA(본서 범위 밖)** 는 다루지 않음. $\vec a\cdot\vec b=\lvert\vec a\rvert\lvert\vec b\rvert\cos\theta$ 의 *증명*(코사인법칙)은 보류(정사영 직관으로 의미만 수용). ∂·∇ 여전히 미사용. → **Part Ⅰ(기하·선형대수 블록) 완결**, 다음은 칼럼① 오일러 또는 Part Ⅱ Ch.12 확률.

> **Ch.10 미지로 남긴 것 (의도적)**: **벡터·내적·정사영·코사인 유사도는 여전히 Ch.11** — Ch.10은 cos를 "단위원 점의 가로 좌표"로만 가르치고, "두 방향이 얼마나 같은가(코사인 유사도)"는 *이름만 예고*(11강). 화살표/점은 '벡터'라 부르지 않음(쓰면 [미래당김]). **위치 인코딩은 Ch.20** — Ch.10은 "회전을 곱셈으로"까지만, 트랜스포머 적용은 이름만. **오일러 항등식 $e^{i\pi}+1=0$ 은 칼럼①**(10강은 $\cos\theta+i\sin\theta$ 회전까지만, $e^{i\theta}$ 형태·증명 없음 — 6강 e와 만나는 건 칼럼에서). ∂·∇ 여전히 미사용.

> **Ch.9 미지로 남긴 것 (의도적)**: **벡터·내적·정사영·코사인 유사도는 여전히 Ch.11** — Ch.9 행렬곱은 "행·열 짝지어 곱해 더함"이라는 *기계적 절차*로만 가르치고 '벡터/내적'이라는 용어·개념은 쓰지 않았다(쓰면 [미래당김]). **편향 $b$·뉴런 정식은 Ch.16**(Ch.9는 $y=Wx+b$를 '섞고 더하기' 직관 + AI 연결로만, $b$는 8강 맛보기 연장). **전치가 역전파에 쓰이는 이유는 Ch.19**(Ch.9는 "행↔열 뒤집기" 기계만). ∂ 여전히 미사용.

> **Ch.8 미지로 남긴 것 (의도적, v3 갱신)**: 벡터·내적(Ch.11), **그래디언트 기호 $\nabla L$ + 다변수 갱신식 $\theta\leftarrow\theta-\eta\nabla L$ + 편미분 ∂ 계산 = 전부 Ch.19**(8강은 이 셋을 *기호 없는 일상어 직관*까지만; 편미분·그래디언트는 이름만 언급, θ식은 8강에서 삭제), ∇가 최급상승 방향인 *증명*(11강 보류). → ∇·θ·∂ 기호를 쓰거나 다변수 미분을 손계산하면 Ch.9~18에서 [미래당김] 위반.

**Ch.8 '맛보기' 추가 결정 — 2026-06-29 (저자 승인)**: Ch.8 기술 파트에 `맛보기 : 진짜 예측을 한 걸음 학습시키기` 절 신설. $\hat y=wx+b$ → 손실 $(y-\hat y)^2$ → 7강 체인룰로 $w,b$ 각각 1스텝 미분 → 갱신 → 수치로 예측 개선($30\to35.1$)까지 **딱 한 걸음**만 보여줌(저자 손글씨 SGD 출처). **당겨온 것과 처리**:
> - **MSE 손실 $(y-\hat y)^2$** — 이름만("정식은 15강"), 제곱 이유 직관만. → Ch.15에서 [미래당김] 아님(맛보기 선행 인지).
> - **예측식 $wx+b$(뉴런)** — 이름만("정식은 16강"), '직선으로 점수 추측' 직관만. → Ch.16에서 [미래당김] 아님.
> - **편미분** — 단어 1회 + "상수로 묶고 미분"까지만. **∂ 기호·정식 손계산·반복 루프는 여전히 Ch.19로 이연**(맛보기는 $d/dw,\ d/db$ 보통미분 표기 사용, ∂ 미사용 유지).
> - 다회 반복·전체 학습 루프·∂ 정식 전개 = **Ch.19 역전파**에서 손글씨 전체대로 완성(클라이맥스 보존). **손글씨 원본**: `planning/refs/ch19-역전파-손글씨-SGD.png` (Lightshot 임시본을 프로젝트로 영구 복사). 집필 지침은 최신 outline의 Ch.19 항목 ★집필 지침 줄 참조.

> ⚠️ **SUPERSEDED — 2026-06-29 (deep-talk, 저자 결정 B)**: 위 '맛보기 추가 결정'의 *worked example*(예측 30→손실 2601→체인룰로 $dL/dw=-204$ 등→갱신→35.1)은 **삭제하고 비유 한 문단으로 축소**함. 사유: 추상 갱신식 $\theta\leftarrow\theta-\eta\nabla L$(③)이 계산 불가능한 채 노출돼 혼란 → ③·∇ 삭제 + 경사하강법은 "토이 1변수 직접 굴리기(3·4단계)"로 8강에 한정, **예측→손실→손실미분→$w,b$갱신의 정식 계산은 19강이 처음부터 전개**(8강은 비유까지). 따라서:

## ✅ Ch.8 (경사하강) 점검·결정 — 2026-06-29

**독자가 Ch.8 진입 시점에 아는 것**: Ch.1~7 = **오직 1변수 미적분**.

**조사 결정 (편미분 이연, 옵션 b):** "편미분이 Ch.8에 꼭 필요한가"를 조사한 결과 — 표준 직관-우선 입문(3B1B·Andrew Ng·fast.ai)은 예외 없이 **1변수 경사하강을 먼저** 가르치고 편미분 계산은 뒤로 미룬다. 1변수만 아는 독자도 "기울기 부호 → 반대로 한 걸음 / f'(x)=0을 못 풀어 굴러내려간다"로 Ch.8을 완결 이해할 수 있다. 그래서 Ch.8에서 **편미분 ∂ 손계산과 예제 3($w_1^2+w_2^2$)을 제거**하고:
1. 그래디언트는 **기호 없이 직관으로만** — "변수마다 제 기울기가 하나씩, 그 묶음의 반대로 모두 함께 한 걸음"(∇L 기호·θ 갱신식은 8강에서 삭제, 정식은 19강 — v3 supersede 참조).
2. 편미분은 **이름만 한 번 언급**하고 "계산은 19강에서"로 이연(∂ 기호·손계산 없음).
3. "직접 손풀이" 정체성은 **1변수 worked example(예제 1·2)** 로 충족.
4. "벡터" 단어·∂ 표기 미사용 유지(벡터 Ch.11, 다변수 미분 Ch.19 전).

> 교훈(가/나 섹션): **책 자신의 선수 순서가 곧 필터다.** 좋아 보이는 빌드업도 뒤 챕터(벡터·다변수)를 당겨오면 [미래당김] 위반 — 직관으로 닫고 정식은 제자리에서. 표준 입문 조사로 "빼도 이해된다"를 확인한 뒤 결정.
