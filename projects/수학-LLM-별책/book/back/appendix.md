## 부록 B · 연습문제 해답

먼저 혼자 풀어 본 뒤 맞춰 보세요. 답만 외우지 말고, 풀이 과정이 어디서 막혔는지 확인하는 게 중요합니다.

### Ch.1 · 최솟값

**1.** $x^2-6x+10 = (x-3)^2 - 9 + 10 = (x-3)^2 + 1$. 최솟값 **1**, 그때 $x=3$.
**2.** $2x^2+8x+5 = 2(x^2+4x)+5 = 2(x+2)^2 - 8 + 5 = 2(x+2)^2 - 3$. 최솟값 **−3**($x=-2$).
**3.** $L(w)=(w-3)^2+1$ 은 $w=3$ 에서 제곱항이 0이 되어 최소. 따라서 $w=\mathbf{3}$, 그때 손실 **1**.

### Ch.2 · 지수·제곱근

**1.** $\dfrac{3^5\times 3^2}{3^4} = 3^{5+2-4} = 3^3 = \mathbf{27}$.
**2.** $16^{3/4} = (16^{1/4})^3 = 2^3 = \mathbf{8}$.
**3.** 거리 $= \sqrt{(5-2)^2+(5-1)^2} = \sqrt{9+16} = \sqrt{25} = \mathbf{5}$.

### Ch.3 · 로그

**1.** $\log_3 81 = \mathbf{4}$ ($3^4=81$).
**2.** $\log_2(4\times 32) = \log_2 4 + \log_2 32 = 2 + 5 = \mathbf{7}$.
**3.** $\log_2 P = \log_2(2^{-1}\cdot 2^{-2}\cdot 2^{-3}) = (-1)+(-2)+(-3) = \mathbf{-6}$.

### Ch.4 · 함수·극한·기울기

**1.** 기울기 $= \dfrac{13-5}{6-2} = \dfrac{8}{4} = \mathbf{2}$.
**2.** $\dfrac{(3+h)^2-9}{h} = \dfrac{9+6h+h^2-9}{h} = \dfrac{6h+h^2}{h} = 6+h$. $h\to0$ 이면 순간기울기 **6**.
**3.** 그릇의 가장 낮은 점에서 접선은 수평이므로 순간기울기는 **0**. (5강에서 $f'(x)=0$ 으로 바닥을 찾는 근거.)

### Ch.5 · 미분

**1.** $f'(x)=3x^2$, $x=2$ 에서 $3\cdot 4 = \mathbf{12}$.
**2.** $f'(x)=2x-6=0 \Rightarrow x=3$. $f(3)=9-18+10=1$. 최솟값 **1**($x=3$) — 1강 답과 일치.
**3.** $L'(w)=2w-2=0 \Rightarrow w=\mathbf{1}$. (그때 손실 $L(1)=1-2+5=4$.)

### Ch.6 · e와 ln

**1.** $\left(1+\tfrac{1}{1000}\right)^{1000} \approx 2.717$ — $e\approx 2.71828$ 에 더 가까워집니다. **네.**
**2.** $(2^x)' = 2^x\ln 2$. $x=0$ 에서 $2^0\ln 2 = 1\times 0.693 = \mathbf{0.693}$ — 표의 값과 일치.
**3.** $f'(x)=3e^x$. $x=0$ 에서 $3e^0 = 3\times 1 = \mathbf{3}$.

### Ch.7 · 체인룰

**1.** 속 $u=2x$($u'=2$), 겉 $e^u$. $f'(x) = e^{2x}\cdot 2 = \mathbf{2e^{2x}}$.
**2.** $f'(x) = 4(3x-1)^3\cdot 3 = \mathbf{12(3x-1)^3}$.
**3.** $u=x^2+1$($u'=2x$). $f'(x) = \dfrac{u'}{u} = \mathbf{\dfrac{2x}{x^2+1}}$.

### Ch.8 · 경사하강법

공통: $x_{n+1} = x_n - \eta f'(x_n)$.
**1.** $f'(x)=2x-4$. $x_2=0.72$ 에서 $x_3 = 0.72 - 0.1(2\cdot 0.72 - 4) = 0.72 - 0.1(-2.56) = \mathbf{0.976}$.
**2.** $f'(x)=2x$, $\eta=0.5$. $x_1 = 3 - 0.5(2\cdot 3) = 3-3 = 0$. $x_2 = 0 - 0.5(0) = 0$. **한 걸음에 바닥 $x=0$ 에 도착해 그대로 머뭅니다.**
**3.** $f'(x)=2x-4$, $\eta=1$. $x_1 = 0 - 1(2\cdot 0 - 4) = 4$. $x_2 = 4 - 1(2\cdot 4 - 4) = 4-4 = 0$. **0과 4를 같은 폭으로 영원히 왕복**(수렴도 발산도 아닌 경계).

### Ch.9 · 행렬

**1.** $\begin{bmatrix}1&2\\0&3\end{bmatrix}\begin{bmatrix}2\\1\end{bmatrix} = \begin{bmatrix}1\cdot2+2\cdot1\\0\cdot2+3\cdot1\end{bmatrix} = \begin{bmatrix}\mathbf{4}\\\mathbf{3}\end{bmatrix}$.
**2.** $\begin{bmatrix}4\\3\end{bmatrix}+\begin{bmatrix}-1\\2\end{bmatrix} = \begin{bmatrix}\mathbf{3}\\\mathbf{5}\end{bmatrix}$.
**3.** $A^\top = \begin{bmatrix}2&1\\0&3\\5&4\end{bmatrix}$. 크기가 $2\times 3 \to \mathbf{3\times 2}$ 로 바뀝니다(행과 열이 뒤바뀜).
**4.** 왼쪽의 열 수(2)와 오른쪽의 행 수(2)가 같으므로 **곱할 수 있고**, 결과 크기는 $\mathbf{3\times 1}$.

### Ch.10 · 삼각·복소

**1.** $\sin^2 60^\circ = 1 - \cos^2 60^\circ = 1 - \tfrac14 = \tfrac34$. 1사분면이라 양수이므로 $\sin 60^\circ = \dfrac{\sqrt3}{2} \approx \mathbf{0.866}$.
**2.** $(1+4i)i = i + 4i^2 = i - 4 = \mathbf{-4+i}$. 점으로는 $(-4,\ 1)$.
**3.** 출발 $(1,4)$: $\sqrt{1^2+4^2}=\sqrt{17}$. 결과 $(-4,1)$: $\sqrt{(-4)^2+1^2}=\sqrt{17}$. **둘 다 $\sqrt{17}$** — 회전이 길이를 바꾸지 않았습니다.

### Ch.11 · 벡터·내적

**1.** $\vec a\cdot\vec b = 1\cdot3 + 2\cdot1 = \mathbf{5}$.
**2.** $\lvert\vec a\rvert = \sqrt{6^2+8^2} = \sqrt{100} = \mathbf{10}$.
**3.** $\vec a\cdot\vec b = 1\cdot0+0\cdot5 = 0$, 코사인 유사도 $= \dfrac{0}{1\times5} = \mathbf{0}$. 두 벡터는 **직각(수직)** — 방향이 전혀 안 겹칩니다.
**4.** 코사인 유사도가 음수라는 뜻이므로, 두 화살표는 **둔각(90°보다 큰 각)** 을 이룹니다 — 대체로 **반대 방향**을 향합니다.

### Ch.12 · 확률·정규분포

**1.** $P(\text{4,5,6}) = \dfrac{3}{6} = \mathbf{\tfrac12}$.
**2.** $P(C) = 1 - 0.5 - 0.2 = \mathbf{0.3}$.
**3.** $45=\mu-\sigma$, $75=\mu+\sigma$ 이므로 $\pm1\sigma$ 구간 → **약 68%**.
**4.** $90 = \mu+2\sigma$. $\pm2\sigma$ 안이 95%이니 바깥 양 꼬리가 5%, 한쪽(90 초과)은 그 절반 → **약 2.5%**.

### Ch.13 · 평균·분산·표준편차

**1.** $\bar x = \dfrac{2+4+6+8}{4} = \dfrac{20}{4} = \mathbf{5}$.
**2.** 편차 $-3,\ -1,\ 1,\ 3$. 합 $-3-1+1+3 = \mathbf{0}$ — 상쇄됨을 확인.
**3.** 분산$(\div n)=\dfrac{9+1+1+9}{4}=\dfrac{20}{4}=\mathbf{5}$, 표준편차 $\sqrt{5}\approx\mathbf{2.24}$.
**4.** $\div(n-1)=\dfrac{20}{3}\approx\mathbf{6.67}$ — 전체로 본 분산(5)보다 **큽니다**.

### Ch.14 · 공분산·상관

**1.** 공분산이 음수 → 두 변수가 **반대로** 움직입니다(한쪽이 크면 다른 쪽은 작은 경향).
**2.** 곱 $(-1)(-2),\ 0,\ (1)(2) = 2,\ 0,\ 2$. 합 4. $\mathrm{Cov} = \dfrac{4}{3} \approx \mathbf{1.33}$.
**3.** $s_x s_y = \sqrt{\tfrac23}\cdot\sqrt{\tfrac83} = \sqrt{\tfrac{16}{9}} = \tfrac43$. $r = \dfrac{4/3}{4/3} = \mathbf{1}$ — 완벽한 양의 상관.
**4.** $\mathrm{Cov}(x,x)$ 는 곧 $x$ 의 **분산**입니다(13강).

### Ch.15 · MSE

**1.** 오차 $2,\ -2$ → 제곱 $4,\ 4$ → $\text{MSE} = \dfrac{4+4}{2} = \mathbf{4}$.
**2.** 오차가 모두 0이므로 $\text{MSE} = \mathbf{0}$ — **전혀 틀리지 않은 완벽한 예측**.
**3.** ① 양수·음수 오차의 **부호 상쇄를 막는다**, ② **큰 오차에 더 큰 벌점**을 준다.
**4.** 포물선의 **꼭짓점(가장 낮은 바닥)**.

### Ch.16 · 뉴런

**1.** $y = -3\times 2 + 5 = -6+5 = \mathbf{-1}$.
**2.** $y = 4+5+6+0 = \mathbf{15}$. 가중치가 모두 1, 편향 0이라 **입력을 그냥 다 더하는** 셈입니다.
**3.** $b$ 를 3 늘리면 출력도 **3 늘어납니다**(직선 전체가 위로 평행이동). $w$ 는 출력의 **기울기**(입력 변화에 대한 민감도)를 정합니다.
**4.** $y = 0.2\times10 + 0.8\times5 - 1 = 2+4-1 = \mathbf{5}$.

### Ch.17 · 활성화함수

**1.** $\text{ReLU}(5)=\mathbf{5}$, $\text{ReLU}(-1)=\mathbf{0}$, $\text{ReLU}(0)=\mathbf{0}$.
**2.** $\sigma(0)=\mathbf{0.5}$. $x$ 가 아주 큰 양수면 $\sigma(x)\to\mathbf{1}$.
**3.** 점수가 같으므로 $\text{softmax}(1,1) = \left(\dfrac{e}{e+e},\ \dfrac{e}{e+e}\right) = \mathbf{(0.5,\ 0.5)}$ — 반반.
**4.** 직선만 쌓으면 전체도 직선이라 곡선·복잡한 패턴을 표현 못 합니다. 활성화함수가 출력을 **휘어(비선형)** 주어 곡선 표현을 가능케 합니다.

### Ch.18 · 순전파

**1.** 위첨자 $(l)$ 은 **"몇 번째 층인가"** 를 나타내는 번호입니다. **거듭제곱이 아닙니다.**
**2.** ① 행렬곱 $W\vec a$(9강) → ② 편향 더하기 $+\vec b$(16강) → ③ 활성화 $f(\cdot)$(17강).
**3.** $W\vec x = \begin{bmatrix}1&1\\1&-1\end{bmatrix}\begin{bmatrix}2\\0\end{bmatrix} = \begin{bmatrix}2\\2\end{bmatrix}$, 편향 0, ReLU 통과 → $\vec a^{(1)} = \mathbf{(2,\ 2)}$.
**4.** 순전파는 **입력 → 출력** 방향. (역전파는 그 반대인 출력 → 입력 방향입니다.)

### Ch.19 · 역전파

**1.** **편미분**은 여러 변수 중 다른 것을 상수로 두고 한 변수로만 미분합니다. 보통 미분은 변수가 하나뿐일 때 씁니다(그래서 기호도 $\partial$ vs $d$).
**2.** $\hat y = 1\cdot4+0 = 4$, 오차 $y-\hat y = 10-4 = 6$. $\dfrac{\partial L}{\partial w} = -2x(y-\hat y) = -2\cdot4\cdot6 = \mathbf{-48}$, $\dfrac{\partial L}{\partial b} = -2(y-\hat y) = \mathbf{-12}$.
**3.** $w \leftarrow 1 - 0.1(-48) = 1+4.8 = \mathbf{5.8}$. $b \leftarrow 0 - 0.1(-12) = \mathbf{1.2}$.
**4.** $\theta \leftarrow \theta - \eta\nabla L$. 이는 8강의 **"변수마다 기울기를 재서 그 묶음의 반대로 모두 함께 한 걸음"** 이라는 비유를 정식화한 것입니다.

### Ch.20 · 어텐션

**1.** **내적**(Ch.11) — Query와 Key를 짝지어 곱해 더해 닮음 점수를 냅니다.
**2.** **softmax**(Ch.17) — 점수를 합이 1인 가중치(확률분포)로 바꿉니다.
**3.** $\vec q\cdot\vec k_1 = 2\cdot1+0\cdot0 = 2$, $\vec q\cdot\vec k_2 = 2\cdot0+0\cdot3 = 0$. 점수가 더 높은 $\vec k_1$ 을 **더 주목**합니다.
**4.** 출력 $= 0.5(2,0) + 0.5(0,4) = (1,0)+(0,2) = \mathbf{(1,\ 2)}$.
