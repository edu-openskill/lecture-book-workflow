# answers — 수학·LLM 별책

## STEP 1: 씨앗 (진행 중)

- **트랙**: concept (목차/주제)
- **주제 (Q1)**: "AI/LLM을 이해하기 위한, 가장 쉬운 수학" — **이해시키는 개념서(A) + AI에 쓰이는 부분만(C) 결합**.
  - 핵심 규칙(저자 확정): **수학책이므로 비유만으로 끝내지 않는다.** 매 강은 *비유로 직관을 열고, 기술 파트에서 수식을 정식 전개(LaTeX)한 뒤 계산 예제를 직접 풀어보며 닫는다.* 예외 없음.
  - → 책 유형 = 스토리텔링 개념서 + 정식 수식/계산. (복습서·드릴이 아니라, 비유→수식→풀이의 3단)
  - 한 줄 컨셉(목차에서): "AI가 배운다 = 손실의 최솟값을 찾는다. 그 한 문장을 수학으로 분해한다."
- **집필 동기 (Q2)**: C+D 결합 — (C) 입문자가 'AI 때문에' 수학을 다시 펴는데 딱 필요한 부분만 짚어주는 안내서가 없어서, (D) 비유로 '이해한 척'에서 멈추지 않고 직접 풀어 진짜 자기 것이 되게 하려고.
- **저자 경험 / 실패담 (Q3)**: **AI에 필요도 없는 수학까지 전부, 그것도 비효율적으로 공부하며 시간을 크게 낭비함.** → 이 책은 그 "최단거리 지도". 비효율을 직접 겪은 사람이 쓰는 권위.
- **차별점**: "AI 수학은 다 할 필요 없다 — 딱 쓰이는 만큼만, 비유로 이해하고 직접 풀어서." (전부 훑는 복습서도 아니고, 비유만 하는 입문서도 아님)
- **핵심 메시지 (Q4 = D)**: **"필요한 만큼만, 대신 직접 풀어서 확실히."** ← 이후 모든 결정의 의도 필터.
- **책의 의도 (Q5, D로부터)**: "이해"에서 멈추지 않고 **"직접 풀 수 있게"** 까지. (each 강: 비유 직관 → 수식 전개 → 손으로 풀어보기)
- **의도 밖 범위 (Q6)** — 필터 D 적용:
  - **11강 오일러 항등식 → 정식 강에서 빼고 "쉬어가는 칼럼"으로** 짧게 (AI 직접 쓰임 없음, 통합의 쾌감용 / 잎사귀 노드라 도미노 없음).
  - **5강 복소평면 → 4강 삼각함수에 편입**(합본). (주 용도가 11강 무대 깔기였음)
  - **13-4 신뢰구간·p-value → 삭제** (연구방법론/평가 쪽, 핵심 경로 밖).
  - **13-3 모집단 vs 표본 → 삭제** (통계 일반론, LLM 직결도 낮음).
  - 절대 못 빼는 것(선수 의존): 13강 벡터/내적/PCA(→14-6 어텐션), 9강 체인룰(→14-5 역전파), 10강 경사하강법.
