4개 레이아웃 비교

레이아웃세로 길이가로 길이흐름 가독성적합 상황
V1 세로 스택길다좁음위→아래 명확전자책·긴 페이지
V2 2열 컴팩트보통보통왼쪽 흐름 + 오른쪽 저장한 페이지 인쇄 · 균형
V3 수평 파이프짧다매우 김시계열 흐름 최강가로 페이지·슬라이드
V4 격자 매트릭스짧다보통한눈 조망완성본 요약 · 인쇄 최적

선택 가이드

  • 전자책·웹 읽기 (스크롤 무제한) → V1 세로 스택
  • 인쇄 한 페이지에 맞춤 + 흐름도 보존 → V2 2열 컴팩트
  • "질문이 들어와서 답변까지" 시계열 강조 → V3 수평 파이프 (책은 세로이지만 가로 전면 페이지로 배치 가능)
  • 완성본을 한 장에 조망 → V4 격자 매트릭스 (§10.10 최종본 특히 좋음)

권장 조합

  • CH02~CH07 §N.last: V2 2열 컴팩트. 한 페이지에 요청 경로 + 저장소를 깔끔히. 챕터가 진행되며 accent 영역이 채워지는 progression 시각화도 유지
  • CH10 §10.10 최종 완성본: V4 격자 매트릭스. 모든 층이 채워진 상태를 한 장에 조망. 마지막 장의 여운
