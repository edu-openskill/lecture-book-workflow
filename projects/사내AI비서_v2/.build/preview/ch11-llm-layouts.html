11.5 LLM 배치 4가지 레이아웃 — 좌우 공백 제거 + 흐름 강화

현재 LLM 박스가 상단 중앙에 놓여 좌우에 큰 공백이 생기는 문제를 해결하는 4가지 접근. 동일 T3 스타일을 베이스로, LLM 박스 위치와 폭만 바꿉니다.

L1. Full-width LLM Bar — 3 모델 가로 나열

LLM 가로 폭 100% 3 모델 column grid 공백 제거 가장 단순

장점: 구조 변경 최소. 좌우 공백 완전 제거. 3 모델 비교 시각. 단점: LLM 박스가 길어져 내용 비중이 커 보일 수 있음.

결정 가이드

레이아웃 공백 해결 흐름 가시성 추천 상황
L1 풀와이드 ✓ 완전 제거 중: 위→아래 기본 구조 가장 안전·단순. 구조 변경 최소
L2 사이드 라벨 △ 일부 (라벨로 채움) 중상: 외부 의존 문맥 추가 LLM 위치 유지하면서 정보 밀도 올림
L3 세로 5행 ★ ✓ zero 최고: 계층적 ↓ 흐름 즉시 읽힘 흐름을 가장 강조하고 싶을 때
L4 4-Column ✓ 상단 공백 zero 중하: 4열이라 직선 흐름 약함 세로 길이 절약 원할 때

추천: L3 (세로 5행 스택). "요청이 위에서 아래로 흐른다"는 데이터 플로우가 즉시 읽히고 공백도 전혀 없음. 안전한 옵션은 L1 (풀와이드) — 구조를 크게 바꾸지 않고 좌우 공백만 제거.
