CH01 재디자인 매칭

CH01이 실제 쓰는 주요 컴포넌트 7종. 전부 현재 상태 유지로 정정됨. 이유는 아래 배너 참조.

CH01은 이번 리팩터 영향이 없습니다.
이번 정리의 축소 대상은 success · warning · danger 상태변수(Utility)뿐입니다. Secondary(accent-warm · 오렌지)는 브랜드 보조 컬러로 승격되어 :::remember · :::tip · .term에서 그대로 유지됩니다. CH01은 상태변수를 쓰지 않아 시각적 변화 0건.

1. :::remember KEEP

"이것만은 기억하자" 박스. 브랜드 Secondary(오렌지) 그대로 유지. 사용자 지정: "기존 컴포넌트를 사용하자".

이것만은 기억하자

  • RAG는 오픈북 시험입니다. 외우지 말고 필요할 때 찾아보게 한다.
  • 환각의 원인은 기억에만 의존하는 구조입니다. 기억 + 근거로 바꾸면 해결.
변경 없음 · h2/strong color: --color-accent-warm-text (유지)

2. :::term-box + .term CHANGED

용어 정의 박스와 본문 내 .term 강조를 Primary(인디고)로 통일. .term-box의 border-left·tb-name도 Info(블루)에서 Primary로 이동. 현재 실제 CSS가 이미 Primary 적용됨 — 아래는 새 스타일 렌더.

사서가 해주는 건 환각(Hallucination)을 막아 주는 일입니다. 정답을 모르니 지어내는 증상.

비유진짜 용어정식 정의
지어내는 증상Hallucination (환각)LLM이 학습 데이터에 없는 사실을 그럴듯하게 생성하는 현상
.term color: --color-accent-warm (오렌지) → --color-accent (인디고)
.term-box border-left: #2563eb (Info) → var(--color-accent) (Primary)
.term-box .tb-name color: --color-info-text → --color-accent-text

3. :::tip KEEP

실험 해석 박스. Secondary 오렌지 배경·텍스트 유지.

RAG 첫 실험에서 생각해 볼 것들
  • 질문과 문서의 거리: 질문 어휘가 문서에 없으면 검색이 못 잡는다.
  • 청크 경계: 정답이 두 청크에 걸쳐 있으면 조각난다.
변경 없음 · tip 본문 color: --color-accent-warm-text (유지)
