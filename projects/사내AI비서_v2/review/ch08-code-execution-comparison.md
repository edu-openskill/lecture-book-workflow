## 교육 목적 부적합 사항

1. **Step 2**: "연차 신청 절차" 쿼리에 육아휴직(d10)이 1위 → 리랭킹이 오히려 관련 문서를 밀어냄. 챕터 서사("병가를 물었는데 출장이 나왔다")와 연결하려면 병가 쿼리로 변경 필요
2. **Step 1 이미지**: step 1-1(크기 비교) 캡처를 step 1-3(전략 비교) 자리에 사용
3. **Step 3 이미지**: 해상도 너무 낮아 텍스트 판독 불가
4. **Step 3 결과**: 3위에 "개인 USB" 문서가 올라옴 (BM25 "신청" 키워드 매칭) — 하이브리드의 한계를 보여주지만 교육 메시지가 흐려질 수 있음
