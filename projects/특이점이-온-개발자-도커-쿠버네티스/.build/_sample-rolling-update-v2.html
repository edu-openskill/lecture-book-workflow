롤링 업데이트 다이어그램 — 외부 리서치 반영 개선안 비교

개선 A — Pod 상태 라벨 추가 (Ready / 생성 중 / Terminating)

새 Pod는 점선·빈 원(생성 중) → 채운 원(Running)으로, 기존 Pod는 점선·회색(Terminating)으로. 터미널 그림의 STATUS와 용어 일치.

개선 A — 상태 라벨(Ready/생성 중/Terminating)로 터미널과 연결

개선 B — "항상 4개 Ready 유지" 불변선 추가

기존 샘플(상태 라벨 없음)에, 가용 Pod가 4개 밑으로 내려가지 않는다는 무중단 기준선을 명시.

개선 B — 무중단 기준선(Ready ≥ 4)으로 maxUnavailable 0의 의미 강조

개선 C — A + B 둘 다

상태 라벨 + 무중단 기준선. 정보량 최대.

개선 C — 상태 라벨 + 무중단 기준선 (정보량 최대)
