그림 5-13 — 옵션 B (두 단계 분리) 샘플 v1

단계 1 · 시작 시 또는 Ingress 리소스 변경 시
Ingress Controller가 Ingress 리소스를 한 번 읽어 내부 라우팅 설정을 갱신합니다 (요청 처리와는 무관한 시점)
단계 2 · 외부 요청 처리 시
Ingress Controller는 내부 라우팅 설정으로 즉시 분기합니다 (Ingress 리소스를 다시 읽지 않음)
그림 5-13. Ingress Controller는 시작·변경 시 Ingress 리소스를 한 번 읽어 내부 라우팅 설정을 갱신하고, 요청 처리 시에는 내부 설정으로 즉시 분기합니다
