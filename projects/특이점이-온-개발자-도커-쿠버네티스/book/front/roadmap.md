# 이 책의 로드맵

오픈이의 여정은 세 단계로 이루어져 있습니다. 각 장이 끝날 때마다 오픈이의 능력은 한 단계씩 성장합니다.

---

```
 ┌──────────────────────────────────────────────────────────────────┐
 │                                                                  │
 │  "내 컴에서는 됐는데..."                                          │
 │         │                                                        │
 │         ▼                                                        │
 │  ┌─────────────────────────────────────────────┐                 │
 │  │  1장. Docker 이해하기                        │                 │
 │  │  ─────────────────────────────────────────── │                 │
 │  │  상자를 만들고, 물건을 담고, 창고에 보관한다    │                 │
 │  │                                              │                 │
 │  │  ✓ 컨테이너를 실행하고 내부를 탐험한다         │                 │
 │  │  ✓ 내가 만든 이미지를 Dockerhub에 올린다      │                 │
 │  │  ✓ 마운트로 데이터를 안전하게 보존한다         │                 │
 │  └──────────────────────┬──────────────────────┘                 │
 │                         │                                        │
 │              "매번 같은 걸 반복하고 있잖아..."                      │
 │                         │                                        │
 │                         ▼                                        │
 │  ┌─────────────────────────────────────────────┐                 │
 │  │  2장. Docker 다루기                          │                 │
 │  │  ─────────────────────────────────────────── │                 │
 │  │  상자가 여러 개면, 화물선에 한꺼번에 싣는다     │                 │
 │  │                                              │                 │
 │  │  ✓ Dockerfile로 환경 구성을 자동화한다         │                 │
 │  │  ✓ NGINX, Redis, MySQL로 인프라를 구성한다    │                 │
 │  │  ✓ Compose 한 줄로 풀스택 웹사이트를 띄운다    │                 │
 │  └──────────────────────┬──────────────────────┘                 │
 │                         │                                        │
 │              "새벽에 서버가 죽으면 누가 살려주지?"                   │

 │  │  ─────────────────────────────────────────── │                 │

 │  │                                              │                 │

 │  │  ✓ 컨테이너가 죽으면 자동으로 복구된다          │                 │
```

---

| 장 | 관통 비유 | 끝나면 이런 게 보여요 |
|---|---------|-------------------|
| **1장** | 상자를 만들고 창고에 보관 | 내가 만든 Docker 이미지가 Dockerhub에 올라간다 |
| **2장** | 여러 상자를 화물선에 싣기 | `docker-compose up` 한 줄로 풀스택 웹사이트가 뜬다 |
| **3장** | 항구에 관제 시스템 도입 | Minikube 위에서 K8s가 자동으로 서비스를 운영한다 |
