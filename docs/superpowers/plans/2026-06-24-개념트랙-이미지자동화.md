# 개념 트랙 + 이미지 자동화 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 코드 없이 목차/주제만으로 수학·개념서를 쓰는 "개념 트랙"을 `book-workflow` 하네스에 추가하고(기존 코드 트랙 유지), `[GEMINI/IMAGE PROMPT]` 플레이스홀더 이미지 생성을 Codex CLI로 자동화한다.

**Architecture:** 최상위 변수 `트랙`(코드=기본 | 개념)을 도입한다. 이야기 척추(STEP 1 씨앗·STEP 5 이야기파트·STEP 6·7)는 공유하고, 재료 단계(STEP 2·3·4 + STEP 5 기술파트)만 트랙별로 분기한다. 수식은 본문에 LaTeX로 쓰고 빌드 시 Pandoc이 Typst math로 변환한다. 이미지 자동화는 별도 Python 스킬(`image-gen`)로 두 트랙이 공용한다.

**Tech Stack:** Markdown 워크플로우 파일, CLAUDE.md, Claude Code 스킬(SKILL.md), Python 3 (image-gen 스크립트, pytest), Pandoc + Typst (빌드), Codex CLI (이미지 백엔드).

## Global Constraints

- 하네스 설계 철학 유지: 새 개념을 만들지 않는다. `트랙`은 4번째 개념이 아니라 **변수 하나**다. (spec §2)
- **코드 트랙은 기본값 = 회귀 없음.** 코드 트랙 동작을 바꾸는 변경 금지. (spec §13-1)
- 문체 상수(스토리텔링·대화체·쉽게·비유필수)는 양 트랙 **동일**. (spec §3)
- 산출물 파일은 절대 덮어쓰지 않고 `-vN` 접미사. (CLAUDE.md 규칙)
- 디렉토리 생성 시 brace expansion(`mkdir -p path/{a,b}`) 금지. 한 줄에 하나씩. (CLAUDE.md 규칙)
- 모든 파일 Markdown, 코드블록에 언어 태그 명시. 이미지는 `<img ... width="720">` 형식. (CLAUDE.md, image.md §5)
- 수식 표기 = **LaTeX**. Typst math 직접 작성 금지. (spec §7)
- 이미지 백엔드 = **Codex CLI**. API 키 사용 금지(구독 로그인). codex.exe는 PATH의 `codex`로 호출(버전 해시 경로 하드코딩 금지). (spec §8.2)
- 작업 단위마다 커밋. 브랜치: `feature/개념트랙-이미지자동화` (이미 생성됨).

```python
def fix_math_typst(text: str) -> str:
    """Pandoc Typst 출력에서 LaTeX 잔여 매크로를 Typst math로 보정."""
    import re
    # \begin{aligned}...\end{aligned} → 줄바꿈 + & 정렬 제거(단순화)
    def _strip_aligned(m):
        body = m.group(1)
        body = body.replace('\\\\', '\\ ').replace('&', '')
        return body
    text = re.sub(r'\\begin\{aligned\}(.*?)\\end\{aligned\}', _strip_aligned, text, flags=re.S)
    # \xrightarrow{f} → arrow.r (라벨 손실 허용)
    text = re.sub(r'\\xrightarrow\{[^}]*\}', ' arrow.r ', text)
    return text
```
`fix_typst_content` 함수 본문 시작부에 한 줄 추가:

```python
def fix_typst_content(text: str) -> str:
    """Pandoc 출력의 Typst 코드를 후처리"""

    # ... 기존 처리 ...
```

```python
#!/usr/bin/env python3
"""이미지 플레이스홀더 스캔 → Codex 생성 → 경로 이동 → <img> 교체."""
import re
from dataclasses import dataclass

@dataclass
class Placeholder:
    id: str
    path: str
    prompt: str
    raw_block: str   # 주석+이미지줄+캡션 전체(교체 대상)
    img_line: str
    caption: str | None

_BLOCK = re.compile(
    r'<!--\s*\[(?:GEMINI|IMAGE) PROMPT:\s*(?P<id>[^\]]+)\]\s*\n'
    r'(?P<body>.*?)\n-->\s*\n'
    r'(?P<img>!\[[^\]]*\]\([^)]+\))'
    r'(?:\s*\n(?P<caption>\*[^\n]+\*))?',
    re.S,
)
_PATH = re.compile(r'^\s*path:\s*(?P<path>\S+)\s*$', re.M)

def scan_placeholders(md_text: str) -> list[Placeholder]:
    out = []
    for m in _BLOCK.finditer(md_text):
        body = m.group('body')
        pm = _PATH.search(body)
        path = pm.group('path') if pm else ''
        # prompt = body에서 path:줄과 Style:줄 제외
        lines = [l for l in body.splitlines()
                 if not l.strip().startswith('path:')]
        prompt = '\n'.join(l for l in lines if l.strip()).strip()
        out.append(Placeholder(
            id=m.group('id').strip(),
            path=path,
            prompt=prompt,
            raw_block=m.group(0),
            img_line=m.group('img'),
            caption=(m.group('caption') or None),
        ))
    return out
```

```python
_SAVED = re.compile(r'Saved to\s+file:///(?P<path>\S+\.png)', re.I)

def parse_codex_image_path(stdout: str) -> str | None:
    m = _SAVED.search(stdout)
    return m.group('path') if m else None

def _md_src_from_img_line(img_line: str) -> str:
    m = re.search(r'!\[[^\]]*\]\((?P<src>[^)]+)\)', img_line)
    return m.group('src') if m else ''

def _alt_from_img_line(img_line: str) -> str:
    m = re.search(r'!\[(?P<alt>[^\]]*)\]', img_line)
    return m.group('alt') if m else ''

def replace_placeholder(md_text: str, ph: Placeholder, width: int = 720) -> str:
    src = _md_src_from_img_line(ph.img_line)
    alt = _alt_from_img_line(ph.img_line)
    img_tag = f'<img src="{src}" width="{width}" alt="{alt}">'
    repl = img_tag + (('\n\n' + ph.caption) if ph.caption else '')
    return md_text.replace(ph.raw_block, repl)
```

```python
import shutil
import subprocess
from pathlib import Path

def run_codex_image(prompt: str, codex_cmd: str = "codex") -> str | None:
    """codex exec로 이미지 생성, 저장된 PNG 절대경로 반환. (S1에서 검증된 호출)"""
    full = ("다음 설명으로 이미지 한 장을 생성해서 PNG 파일로 저장해줘:\n" + prompt)
    res = subprocess.run([codex_cmd, "exec", full],
                         capture_output=True, text=True)
    return parse_codex_image_path(res.stdout + "\n" + res.stderr)

def process_file(md_path, project_root, generate=run_codex_image,
                 dry_run: bool = False) -> int:
    md_path = Path(md_path); project_root = Path(project_root)
    text = md_path.read_text(encoding='utf-8')
    phs = scan_placeholders(text)
    count = 0
    for ph in phs:
        if dry_run:
            print(f"[dry-run] {ph.id} → {ph.path}")
            count += 1
            continue
        saved = generate(ph.prompt)
        if not saved:
            print(f"[skip] 생성 실패: {ph.id}")
            continue
        target = project_root / ph.path
        target.parent.mkdir(parents=True, exist_ok=True)
        shutil.move(str(saved), str(target))
        text = replace_placeholder(text, ph)
        count += 1
        print(f"[ok] {ph.id} → {ph.path}")
    md_path.write_text(text, encoding='utf-8')
    return count
```
