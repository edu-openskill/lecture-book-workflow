- 참여: 사용자 + Claude

## 배경
챕터 집필(STEP 5) 시작 직전, Part Ⅰ(Ch.1~11)의 강 순서가 선수관계상 타당한지,
그리고 Ch.11의 PCA가 이 책에 정말 필요한지 검증 요청.

## 핵심 논의

### 1. Ch.1~11 순서 검증
- 위상정렬상 선수관계 **깨짐 0건** (유효).
- 감수 관점 지적 2건:
  - (a) Ch.4(삼각+복소)가 미적분 척추(1→5→6→8→9)의 "섬" → **사용자 반박으로 철회**.
    cos가 내적·코사인유사도에 직접 들어가고, 복소는 오일러·위치인코딩으로 회수됨.
    → 섬이 아니라 **복선(伏線)**.
  - (b) 행렬(10)→벡터(11) 역순 → 의도된 선택으로 유지(내적을 행렬곱에서 먼저 부트스트랩).

### 2. PCA 필요성
- seed.md는 "절대 못 빼는 선수: 벡터·내적·PCA(→어텐션)"로 명기.
- 그러나 도메인 검증: **어텐션은 주성분을 계산하지 않음.** PCA는 어텐션의 선수가 아님.
- 어텐션의 진짜 선수 = 내적·벡터·코사인유사도(·정사영). PCA(주성분/고유벡터)는 아님.
- 이중 충돌: ① 의도 필터("필요한 만큼만") 밖, ② 고유값 챕터가 없어 "직접 풀이" 불가(직관만) → 이 책 정체성과 어긋남.

## 결론 (결정 3건)

1. **PCA·고유값 제거.** Ch.11은 내적·정사영·코사인유사도만. PCA는 향후 별도 ML/DL 수학 책으로 이연.
2. **Ch.4(삼각·복소) → 기하 블록으로 이동(복선 회수 구조).** 내적 바로 앞에 배치.
   - 결과: Part Ⅰ이 [미적분 척추] + [기하·선형대수 블록] 두 덩어리로 정리됨.
3. **Part Ⅰ 전체 재번호 + Part Ⅱ 선수 참조 갱신.**
