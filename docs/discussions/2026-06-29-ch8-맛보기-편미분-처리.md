# Ch.8 맛보기 절 재검토 — v3/B 유지로 귀결 (수학·LLM 별책)

- 날짜: 2026-06-29
- 참여: 사용자 + Claude
- 프로젝트: projects/수학-LLM-별책 (concept 트랙, STEP 5 집필 중)
- 방식: /superpowers:brainstorming
- **결론: 변경 없음. 같은 날 deep-talk로 내린 outline-v3 / known.md '저자 결정 B'를 그대로 유지.**

## 배경

사용자 질문: "8강에 경사하강법이 들어오는 게 맞나? 목차를 뒤로 빼야 하나?
수학을 순서대로 연결해 풀 수 있어야 하고, **중간에 모르는 게 나오면 안 된다.**
아니면 경사하강은 넣되 어려운 수학만 뒤로 빼는 게 맞나?"

## 경과 (정직한 기록)

1. Claude가 **outline-v2를 최신본으로 오인**하고("최신본 자동선택" 규칙 위반) 분석을 쌓아,
   "맛보기를 Ch.8에 **유지하되 편미분 동작을 Ch.5 연장으로 명시 교수**"(A안)를 일단 권장.
2. 그러나 progress.json·outline-v3 확인 결과, **같은 날 deep-talk로 이미 더 단호한 결정(B안)**이
   내려져 있었고 chapters/08·known.md가 **이미 그 결정으로 동기화 완료**된 상태였음:
   - outline-v3 §v3 변경점, Ch.8/Ch.19 ★집필 지침.
   - known.md "Ch.8 미지로 남긴 것(v3 갱신)" + "⚠️ SUPERSEDED — 저자 결정 B" 메모.
   - chapters/08: 맛보기가 이미 *비유 한 문단*, ∇·θ·∂ 0회.
3. 즉 A안은 **stale한 v2에 근거한 잘못된 재도출**이었음. → **A안 철회.**

## 두 안 비교

| | A안 (철회) | **B안 = v3 (유지)** |
|--|-----------|--------------------|
| 8강 맛보기 | worked example 유지(30→35.1 체인룰 계산), 편미분 동작 명시 교수 | **비유 한 문단**(수치·체인룰·∂ 없음) |
| ∇·θ·∂ | "상수로 묶고 미분"을 1회 명시 노출 | **0회** (전부 19강) |
| 손실미분·$w,b$갱신·반복 | 8강에서 1스텝 맛봄 | **19강이 첫 정식 전개** |
| 사용자 원칙 충족 | 회색지대를 '잘 가르쳐' 통과(약) | **회색지대 자체가 없음(가장 깨끗)** |

핵심: 사용자 원칙("중간에 모르는 게 나오면 안 된다")은 **B안이 더 깨끗하게 충족**한다.
8강에 다변수 미분 기계(∇·θ·∂)가 0회이므로, 1변수만 아는 독자가 막힐 지점이 원천적으로 없다.
A안의 "명시 교수"는 회색지대를 우회하는 약한 해법이었다.

## 결론 (결정)

**B안 = outline-v3 유지. 코드/본문/원장 변경 없음.**

- 경사하강법은 Ch.8에 그대로 둔다(★전환점). 단 **토이 1변수**(예제 1·2: η=0.1 수렴 / η=1.2 발산 / 경계 η=1)만 직접 손으로 굴린다.
- 다변수는 **비유로만**(그릇 그림 8-7, 기호 없음).
- 예측·손실·손실미분·$w,b$갱신·편미분 ∂·다회 반복·∇·θ = **전부 Ch.19 역전파에서 첫 정식 전개.**
- 사용자 질문에 대한 답: "경사하강은 넣되 어려운 수학만 뒤로" = 3안과 같은 취지이며, 이미 v3가
  그보다 더 강하게(∇·θ·∂까지 전부 19강으로) 실현해 둠.
